# 🏀 BallCut - 篮球进球集锦生成器

自动检测篮球比赛视频中的进球瞬间，按球员生成个人集锦。

---

## 🚀 快速启动

你可以通过我刚刚为你编写的跨平台启动脚本一键配置环境并启动：

```bash
python start.py
```
*(该脚本会自动检测操作系统、创建虚拟环境、安装所需依赖库，并启动 Web 服务)*

### 访问应用

启动成功后，请在浏览器中访问：**[http://127.0.0.1:8080](http://127.0.0.1:8080)**

---

## ✨ 核心功能

-   **智能检测**：AI 自动分析视频关键帧，定位篮球进球瞬间。
-   **历史记录**：首页自动整理历史处理记录，随时找回之前的进度。
-   **球员归类**：支持在线标注球员，自动合并同名球员的进球集锦。
-   **一键导出**：基于 FFmpeg 高效剪辑，生成流畅的 MP4 集锦视频。

---

## 🛠️ 系统要求

-   **Python**: 3.9+
-   **FFmpeg**: 必需 (用于视频剪辑)
    -   Windows 用户请确保将 `ffmpeg.exe` 路径添加至系统环境变量。
-   **GPU (可选)**：支持 CUDA 可显著提升 AI 检测速度。
